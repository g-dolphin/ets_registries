# Data sources

## EU ETS

The pipeline expects an extracted EUTL directory with:

- `compliance.csv`
- `installation.csv`
- `nace_code.csv`

## UK ETS

- Allocation table: GOV.UK publication “UK ETS allocation table for operators of installations”
- Compliance report: public UK ETS registry report “Compliance Report – Emissions and Surrenders”

## California Cap-and-Trade

- CARB Mandatory Reporting Regulation (MRR): annual “GHG Facility and Entity Emissions” spreadsheets
- Sector allocation totals: CARB Cap-and-Trade Program Data dashboard files
